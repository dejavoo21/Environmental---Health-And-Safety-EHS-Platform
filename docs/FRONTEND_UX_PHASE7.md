## 1. Overview

Phase 7 introduces two major UI modules:
1. **Chemical Management** - Chemical register, SDS management, storage locations
2. **Permit-to-Work** - Permit creation, lifecycle, and permit board

### 1.1 Design Principles

- **Mobile-First**: All screens optimized for tablet use in field
- **Safety-Focused**: High visibility for critical information (hazards, expiry, permit status)
- **Consistent**: Follow existing EHS Portal patterns from Phases 1-6
- **Accessible**: WCAG 2.1 AA compliance
