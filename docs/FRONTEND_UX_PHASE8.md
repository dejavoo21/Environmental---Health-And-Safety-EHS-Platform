## 5. Accessibility

### 5.1 Requirements

| Feature | Implementation |
|---------|----------------|
| Keyboard navigation | All interactive elements focusable |
| Screen reader | ARIA labels for matrix cells and status icons |
| Color contrast | WCAG 2.1 AA compliance |
| Status indication | Never rely on color alone (use icons + text) |
| Focus management | Logical tab order, visible focus rings |

## 7. State Management

### 7.1 Training Store Slices

```typescript
interface TrainingState {
  // Catalogue
  courses: Course[];
  coursesLoading: boolean;
  coursesPagination: Pagination;
  courseFilters: CourseFilters;
  
  // Sessions
  sessions: Session[];
  sessionsLoading: boolean;
  sessionFilters: SessionFilters;
  
  // Assignments
  assignments: Assignment[];
  assignmentsLoading: boolean;
  assignmentFilters: AssignmentFilters;
  
  // My Training (current user)
  myTraining: MyTrainingData | null;
  myTrainingLoading: boolean;
  
  // Matrix
  matrix: MatrixData | null;
  matrixLoading: boolean;
  matrixFilters: MatrixFilters;
  
  // Selected items
  selectedCourse: Course | null;
  selectedSession: Session | null;
  selectedAssignment: Assignment | null;
}
```

### 7.2 Cache Strategy

| Data | Cache Duration | Invalidation |
|------|----------------|--------------|
| Course catalogue | 10 minutes | On CRUD operations |
| Sessions | 5 minutes | On enrollment change |
| My Training | 2 minutes | On completion/assignment |
| Matrix | 5 minutes | On completion/assignment |
| Completions | 5 minutes | On new completion |

## 8. Error Handling

### 8.1 Error States

| Scenario | User Message | Action |
|----------|--------------|--------|
| Session full | "This session is at capacity" | Offer waitlist option |
| Already enrolled | "You are already enrolled in this session" | Show enrollment details |
| Prerequisites not met | "Complete [course] before enrolling" | Link to prerequisite |
| Past due date | "Assignment is overdue" | Allow completion recording |
| Invalid score | "Score must be between 0 and 100" | Inline validation |

### 8.2 Loading States

- Skeleton loaders for cards and tables
- Shimmer effect for matrix grid
- Inline spinners for actions
- Optimistic updates for enrollments
