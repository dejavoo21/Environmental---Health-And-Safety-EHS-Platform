## 1. Overview

Phase 6 introduces new user-facing pages and components for security features, self-service workflows, and theme customisation. This document specifies the UX design for each feature.

**New Pages:**
- Access Request Page (public)
- Admin Access Requests Page
- Forgot Password Page (public)
- Reset Password Page (public)
- Security Centre Page
- 2FA Setup Pages
- Security Audit Log Page (admin)

**New Components:**
- Theme Toggle
- 2FA Prompt Modal
- Password Strength Meter
- Login History Table
- Backup Codes Display
